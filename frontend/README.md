# Frontend - React Framework for the Digital Signage Project

### Frontend Architecture



## Frontend React Vite Project Setup

This document provides necessary instruction on how to setup and run the frontend project on your local machine.

### Prerequisite
- Node.js

#### Running the Project

#### Step 1

Clone the repo or download the project from Github.

```bash
git clone https://github.com/A6Dig/cs410-digital-signage.git
```

#### Step 2

Navigate to the project directory.

```bash
cd cs410-digital-signage\frontend
```

#### Step 3

Build the project.

```bash
npm install
```

#### Step 4

Run the project.

```bash
npm run dev
```

Navigate to http://localhost:5173 and the frontend should load.
